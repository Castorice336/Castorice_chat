# Castorice_chat
A robot bases on Castorice from "Honkai:StarRail"
> “生来带有死亡之触，却温柔地送别每一个生命。”
基于大语言模型（DeepSeek / GLM等）复现《崩坏：星穹铁道》角色“遐蝶”的AI对话程序。支持流式输出、角色性格模拟，可扩展语音与长期记忆。
# ✨ 功能特点
- 🎭 **角色高度还原** —— 通过精细的System Prompt还原遐蝶温柔、克制、带有悲伤隐喻的说话风格
- 🎤 **语音支持（规划中）** —— 计划接入TTS/STT实现完整语音交互
